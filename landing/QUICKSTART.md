# 유니브클래스 랜딩페이지 빠른 시작 가이드

## 설치 및 실행

### 1. 의존성 설치
```bash
cd landing
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

브라우저에서 **http://localhost:5173** 으로 접속하세요.

### 3. 프로덕션 빌드
```bash
npm run build
```

빌드된 파일은 `dist` 폴더에 생성됩니다.

### 4. 빌드 미리보기
```bash
npm run preview
```

## 주요 기능

- ✅ 고급스러운 프리미엄 디자인
- ✅ 반응형 레이아웃 (모바일, 태블릿, 데스크톱)
- ✅ 상담 신청 폼
- ✅ 프로그램 소개 섹션
- ✅ 스무스 스크롤 네비게이션

## 웹 호스팅 배포

빌드 후 `dist` 폴더의 모든 파일을 웹 호스팅 서비스에 업로드하면 됩니다.

### 추천 호스팅 서비스
- Vercel
- Netlify
- GitHub Pages
- AWS S3 + CloudFront
