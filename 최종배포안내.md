## 🚀 지금 바로 배포하기 (5분)

### 1단계: Railway 가입 (1분)
1. https://railway.app 접속
2. "Start a New Project" 클릭
3. GitHub 계정으로 로그인

### 2단계: 프로젝트 배포 (2분)
1. "Deploy from GitHub repo" 클릭
2. `studyground-mvp` 저장소 선택
3. "Deploy Now" 클릭
4. 자동 배포 시작! (2-3분 소요)

### 3단계: 환경 변수 설정 (2분)

배포가 시작되면 Settings → Variables에서 다음 변수들을 추가하세요:

#### 필수 환경 변수

```
NODE_ENV=production
PORT=5001
HOST=0.0.0.0
```

#### JWT_SECRET 생성 및 추가

터미널에서 실행:
```bash
openssl rand -hex 32
```

생성된 값을 Railway에 추가:
```
JWT_SECRET=<생성된_값>
```

#### Google Sheets 설정

**먼저 Google Sheets를 설정해야 합니다!**

1. [GOOGLE_SHEETS_SETUP.md](./GOOGLE_SHEETS_SETUP.md) 파일 참고
2. Google Sheets 스프레드시트 생성
3. 서비스 계정 생성 및 JSON 키 다운로드
4. Railway에 다음 변수 추가:

```
GOOGLE_SHEETS_ID=<스프레드시트_ID>
GOOGLE_SERVICE_ACCOUNT=<JSON_전체_내용>
```

**중요**: `GOOGLE_SERVICE_ACCOUNT`는 JSON 전체를 따옴표 안에 넣어야 합니다.
예: `{"type":"service_account","project_id":"..."}`

### 4단계: 배포 완료 확인

1. Railway 대시보드에서 "Deployments" 탭 확인
2. "Settings" → "Domains"에서 URL 확인
3. 다음 URL로 접속 테스트:
   - 키오스크: `https://your-app.railway.app/kiosk`
   - 관리자: `https://your-app.railway.app/admin`
   - 공용 대시보드: `https://your-app.railway.app/public`

## 🆘 문제 해결

### 배포 실패
- Railway 로그 확인 (Deployments 탭)
- 환경 변수 형식 확인
- Google Sheets 권한 확인

### 접속 불가
- HTTPS URL 사용 확인
- 경로 확인 (`/kiosk`, `/admin`, `/public`)
- Railway 서비스 상태 확인

### Google Sheets 오류
- 서비스 계정 권한 확인
- 스프레드시트 ID 확인
- JSON 형식 확인
